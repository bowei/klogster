klogster — keyboard & feature reference

Panels

＋ Pod
Open the pod browser to add a new log panel.
Panel close (✕)
Close an individual log panel.
Drag tabs
Reorder panels by dragging their header tabs.
Lock / Unlock scroll
Toggle live-tail for a panel. When unlocked, new lines scroll into view automatically.

Filtering

Filter button
Per-panel include/exclude filters using regular expressions. + keeps matching lines, − hides them.
Focus
Add one or more regexp patterns to filter all panels. Lines matching any pattern are shown and the matching text is highlighted. Optional context lines above/below each match are also shown.

Timeline crosshair

Hover over a timestamp
Shows an orange crosshair on all panels aligned to the same moment in time.

State persistence

URL / session storage
Open panels, filters, and focus state are saved automatically and restored on page reload.
